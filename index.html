Citadel

Simplistic Organization

Mission

The goal of Citadel was to make a simple to use interface that allows users to easily contribute to the modifying, sorting, and organizing of student information in a seamless fashion; then allowing the easy exporting and sharing of this data.

Usage

Citadel was intended to be used by the organization Future Business Leaders of America - Phi Beta Lambda (FBLA-PBL), any confusions on the usage of the program can be found on the GitHub project wiki.

Features

Citadel's features were intended to remain very user friendly, while still being able to accomplish tasks quickly and efficiently. A few examples of these features include:

  • A user system with two levels of permissions allowing for the creation and deletion of users, as well as the restriction of lower power users.
  • The creation, combining, or deleting of 'sources' (essentially data sets) which allows for easy categorization and sharing of large portions of data.
  • The ability to easily view the members of a given source as well as searching or filtering the members.
  • The ability to easily create, view (both in detail and summary), delete, or edit members of a given source in an instant.
  • Automatic calculation of the statistics of a given source.
  • The ability to export files in different formats, both locally or through email.
